INSTALLATION
------------

```
#!bash


mkdir build
cd build
cmake ..
make
sudo make install
```



BUILD EXAMPLE
-------------


```
#!bash

mkdir build
cd build
cmake .. -DBUILD_EXAMPLE=ON
make
./../bin/example
```
